## Packages
framer-motion | Essential for futuristic UI animations and smooth transitions
react-player | For playing video clips and full match footage
clsx | For conditional class merging
tailwind-merge | For merging tailwind classes safely
date-fns | For formatting dates and timestamps

## Notes
Authentication is handled via Replit Auth and use-auth hook.
Video playback requires valid file paths or URLs.
UI theme is "Futuristic Sports Analytics" - Dark mode default.
Glassmorphism effects used heavily.
